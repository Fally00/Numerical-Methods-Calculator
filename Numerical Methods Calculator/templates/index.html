Numerical Methods

Bisection Method

Find a root by repeatedly halving an interval [a, b] where f(a)·f(b) < 0.

Doolittle LU Decomposition

Decompose matrix A into Lower (L) and Upper (U) triangular matrices, then solve Ax = b.

False Position (Regula Falsi)

Improved bracketing method using a linear interpolant to estimate the root.

Gauss-Seidel Method

Iterative method for solving linear systems Ax = b using updated values immediately.

Jacobi Method

Iterative method for solving linear systems using previous iteration values only.

Newton-Raphson Method

Fast root-finding using the function and its derivative for tangent-line approximation.

Thomas Algorithm

Efficient solver for tridiagonal linear systems. Provide three diagonals and the RHS vector.
